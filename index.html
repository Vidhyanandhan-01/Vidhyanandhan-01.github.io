Projects

01 / ASIC
CNN Accelerator — RTL to GDS
7nm + 130nm ASAP7 PDK SKY130 Synopsys DC Cadence Innovus

Fully parallel CNN accelerator in SystemVerilog featuring a custom Zig-Zag dataflow Systolic Array with clock-gated Processing Elements supporting multi-precision bit widths and ReLU. Full ASIC flow: RTL → Synthesis → P&R → STA → GDSII.

▸ 15% reduction in dynamic power via clock gating & datapath width optimization
▸ 20% AMAT reduction using Zig-Zag & circular dataflow patterns
▸ Tapeout in progress on SKY130 with integrated SerDes, DFT (scan)
02 / CPU
RISC-V RV32I Pipelined Processor
SystemVerilog Synopsys SAED32 UVM

Full 5-stage pipeline (IF, ID, EX, MEM, WB) with hazard resolution via forwarding, stall control, and pipeline flush. Added 2-bit dynamic branch predictor, write-back data cache, and virtual memory. Synthesized and laid out with Synopsys IC Compiler.

▸ Up to 35% CPI improvement over baseline configurations
▸ Verified with CRV self-checking testbenches + UVM SVA assertions
03 / ARCH
Hybrid Z-Cache + XOR-Cache LLC
ChampSim gem5 PARSEC

Hybrid cache architecture approximating fully associative performance with 4-way hardware, combining Z-Cache placement and XOR-based compression. Introduced adaptive timestamp skewing to bias LRU and multi-hash candidate expansion for robust eviction.

▸ Evaluated IPC and MPKI on PARSEC benchmarks across varying associativity parameters

7nm Full-Custom Cadence Virtuoso HSPICE

Optimized neuron datapath using K-map logic minimization and MUX-based architecture (4:1 decomposed to cascaded 2:1 transmission gate MUXes). Full custom ASIC flow: schematic → layout → DRC/LVS → PEX → STA, with multi-Vt cells for power/timing.
